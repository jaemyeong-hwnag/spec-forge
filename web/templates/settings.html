Notion 연동 설정 방법
  1. notion.so/my-integrations 접속 → + New integration
  2. 이름 입력 (예: Workflow Spec) → Submit
  3. Internal Integration Secret 복사 → 아래 Token 입력란에 붙여넣기
  4. Notion에서 사용할 페이지 열기 → 우측 상단 … → Connections → 방금 만든 Integration 연결
  5. 메인 화면에서 해당 페이지 URL을 붙여넣어 스펙 생성

Figma (선택)

Figma 연동 설정 방법
  1. figma.com/settings 접속 → Personal access tokens
  2. Generate new token → 이름 입력 → 토큰 복사
  3. 아래 입력란에 붙여넣기

설정하면 스펙 생성 시 Figma 링크를 참조할 수 있습니다.
